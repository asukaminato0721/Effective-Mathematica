# 用数学的思维解决问题

Mathematica 底层对于数学相关内容优化得非常好，所以要有这方面的思维。

比如生成

![](http://latex.codecogs.com/svg.latex?x+2x^2+3x^3+4x^4......)

可以

```Mathematica
Range[0, 5].x^Range[0, 5]
```

注意 `Dot` 的使用。

用纯函数写成 `#.x^# &@Range[0, 5]`

---

list 里两两比对，可以 `Subsets[list,{2}]`
