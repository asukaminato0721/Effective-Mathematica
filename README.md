# Effective-Mathematica

一本想要致敬 Effective Python 与 Effective C++ 的小书

---

Mathematica 是一门非常灵活的语言，但是初学者经常会走弯路。总结一些经验。

这里的 Effective 有多重函数

1. 运行效率
2. 写代码效率
3. 查问题效率
